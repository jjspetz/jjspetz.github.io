Other Projects


See the Pen FCC: Kid's Calculator by JJ Spetseris (@jjspetz) on CodePen.

This is a javascript calculator I made while working through freeCodeCamp's front-end course. I made this particular project while I was teaching so the design was inspired by my students' calculators and their lust for stickers. I used jQuery extensively on this project. I learned a lot about the potential perils of accepting user input and some of the limitations JavaScript has with floating point math. I decided to handle all the displays and storage of variables as strings that could be passed around from function to function. This also allowed me to use regular expressions to manipulate the string. This led to the challenge of how to do the actual math. 1 + 1 as a string equals 11 not 2, so i eventually turned to the eval function. The simplicity of this method outweighed security concerns, since users can only enter numbers and a limited number of math equations.
